Mina svårigheter i det föregående projektet var javascript och att koppla ihop java ,css och html med varandra. Jag hade alltså svårt för class och ID taggar och sedan att koppla ihop html och javascript. Det var svårt att få en speciell knapp i html att reagera på något som jag gjort i javascript. Feedbacken jag fick ifrån mitt förra projekt var att jag skulle undvika radbrytning, skriva all kod i gemener och ha ett alt attribut på alla bilder. Vilka ramar (förutsättningar, hinder, möjligheter) finns i detta projekt?

Jag har bra förutsättningar för det projekt jag planerar att göra. Jag har goda kunskaper om att koda en webbsida så jag kommer inte stöta på några större problem. Ifall jag stöter på ett hinder så kommer det nog ha att göra med grids i css eller något i javascript t.ex. att man klickar på en knapp sedan tas man vidare till en annan html sida. Det finns vissa bilder som jag inte kommer kunna använda på grund av användarrättigheter men jag kan jobba runt detta genom att istället genom att skriva text och lägga in en annan bild som är relaterad till det budskap jag förmedlar i texten. Jag har tillåtelse att använda alla andra bilder och liknande som jag planerar att använda i mitt projekt. Jag har många möjligheter för detta projekt. Det finns mycket som jag kan göra och lägga in i webbsidan så jag tror att det inte kommer vara något problem med innehållet (för lite).

Vilka kunskaper har jag redan som jag kan använda mig av i detta projekt?

Jag har kunskaper om i princip allting gällande html och css. Jag har även kunskaper om hur en snygg och smidig layout ser ut och vilka färger som passar tillsammans. Jag har kunskaper om whitespace och hur man kan använda sig av den. Jag ska kombinera alla dessa kunskaper jag har och använda dem i detta projekt. Jag kan även göra en hamburgar meny som jag planerar att ha med.
